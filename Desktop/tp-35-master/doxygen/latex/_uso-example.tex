\doxysection{Uso}
Função principal do programa.\+Realiza a inicialização do programa, lendo os dados de funcionários, ementas e escolhas a partir de ficheiros, e apresenta as informações para o utilizador.

As principais etapas do programa são\+:
\begin{DoxyItemize}
\item Ler dados de ficheiros
\item Verificar erros durante a leitura
\item Mostrar listas detalhadas
\item Apresentar refeições por dia
\end{DoxyItemize}

\begin{DoxyReturn}{Returns}
int Retorna 0 se o programa for executado com sucesso, ou 1 em caso de erro.
\end{DoxyReturn}
\+: O programa espera que os ficheiros estejam organizados como no exemplo\+:
\begin{DoxyItemize}
\item Funcionários\+: \char`\"{}./data/funcionarios.\+txt\char`\"{}
\item Ementas\+: \char`\"{}./data/ementas.\+txt\char`\"{}
\item Escolhas\+: \char`\"{}./data/escolhas.\+txt\char`\"{}
\end{DoxyItemize}
